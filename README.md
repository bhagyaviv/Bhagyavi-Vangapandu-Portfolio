# ⚡ Bhagyavi Vangapandu | Developer Portfolio

A futuristic, premium, and fully responsive personal portfolio website showcasing expertise in **Artificial Intelligence (AI), Generative AI, Cloud Computing, and Full Stack Web Development**. 

Built with React.js, Tailwind CSS (v4), Framer Motion, and Lucide React icons.

## 🚀 Live Preview & Repository
- **GitHub Repository**: [https://github.com/bhagyaviv/Bhagyavi-Vangapandu-Portfolio](https://github.com/bhagyaviv/Bhagyavi-Vangapandu-Portfolio)
- **Local Dev Host**: `http://localhost:5173/`

---

## 🎨 Design Theme & Key Features
- **AI-Inspired Dark Mode Aesthetic**: Deep space background with glowing violet, blue, and cyan accents.
- **Startup Terminal Simulation**: An interactive loading terminal in the Hero section showing startup profiles and system parameters.
- **Glassmorphic Interfaces**: Clean cards using translucent overlays (`backdrop-blur`) and custom border hover transitions.
- **Interactive Tech Arsenal**: Sidebar tab panels for categorizing core languages, AI/ML models, cloud architectures, developer tools, and core computer science concepts with custom animation meters.
- **Verified Credentials Portal**: Modern grid highlighting AWS Certified Cloud Practitioner, Oracle Cloud Infrastructure AI Associate, and GitHub Copilot certifications with instant link verifications.
- **Responsive Navigation**: Features active scroll intersection observers, desktop indicators, and mobile overlay drawer navigation.
- **Contact & Feedback**: Responsive contact cards mapped to verified emails, direct messaging, and social profiles.

## ⚙️ Local Development Setup

To run this project locally, ensure you have **Node.js** installed on your system.

### 1. Clone the repository
```bash
git clone https://github.com/bhagyaviv/Bhagyavi-Vangapandu-Portfolio.git
cd Bhagyavi-Vangapandu-Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the local development server
```bash
npm run dev
```
Open **[http://localhost:5173/](http://localhost:5173/)** in your web browser.

### 4. Build for production
```bash
npm run build
```
Generates optimized static chunks inside the `dist/` directory, ready to deploy to services like Vercel, Netlify, or GitHub Pages.
